# 什么是游戏

游戏是艺术吗？
游戏当然是艺术，并且这是属于现代人的艺术。因为艺术的本质是给人体验，游戏无疑满足这一点。
那么为什么是游戏而不是小说，或者是绘画，游戏最独特的地方在哪里？游戏是以交互为核心带给人体验的艺术。
