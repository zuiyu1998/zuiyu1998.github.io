# 总体流程

玩家通过关卡收集散落的日炎碎片，并在最终关卡合成太阳，终结这个漫漫长夜。
玩家使用图鉴建立建筑，抵御来犯的敌人。建筑可以升级和镶嵌。回合内剩余的资源可以回收。

# 建筑

建筑是关卡中玩家可以操作的主要对象。

- 升级
- 镶嵌
- 消除

建筑本身的功能。

- 可以生产，每隔一定时间生成一个或多个资源。
  生产功能可配置，资源种类可新增
- 也可以攻击。
