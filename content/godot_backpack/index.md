# 背包系统

背包用来存储物体。
背包和物体，这是整个背包系统中最重要的两个抽象。

背包是一个数据容器，它支持的功能如下:

- 添加物品
- 取出物体
- 使用物体
- 分类筛选
- 背包之间物体的交互

背包系统中显示信息的是物体。
物体通常有多种类型。这里的共性是有数据要去显示。要显示的数据是多变的，因此要处于外层。
每个物体要显示的数据如下:

- 物体的 icon
- 物体的说明
- 物体的背景说明
- 物体的备注
- 物体的类别
- 物体的名称
- 物体的 id
