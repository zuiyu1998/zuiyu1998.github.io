# 概述

技能系统是游戏系统中常见的功能。它是如此的常见，以至于所有人习以为常。技能系统描述了人物和外界的交互关系，是成长体验最好的载体。

# 需求

1. 对外界的交互
   1. 对人物的交互
   2. 对物体的交互
   3. 对环境的交互
   4. 等其他

2. 交互能力取决于人物本身的属性

# 功能

1. 技能释放辅助
2. 执行交互
   1. 计算交互的数值
   2. 对交互的双方应用效果。
