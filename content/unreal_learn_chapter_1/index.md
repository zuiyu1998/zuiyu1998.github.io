# GameMode

GameMode 下面有 6 个比较重要的值。

- 玩家
- ui
- 玩家控制器
- 游戏状态
- 玩家状态

# Player

玩家是一个 pawn 的子类。GameMode 中的 Default Pawn 就是玩家的实例。因此使用的时候需要将 Pawn 转换成 Player。
