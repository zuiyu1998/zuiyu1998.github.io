# 状态机

状态机的核心四要素

- 状态 （可持续）
- 事件 （改变状态机的动力）
- 转换 （状态转换的过程）
- 动作 （在状态转换的过程中触发的操作）

# 状态机是什么

状态机应该是一个实体。
状态是一个实体的一个组件。
事件是状态机对外的接口。
转换是状态机根据事件处理组件。
动作是状态机对转换过程中触发的操作。
